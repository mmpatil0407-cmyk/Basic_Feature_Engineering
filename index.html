🎯 Salary Prediction Model

Machine Learning Project using Linear Regression

Technologies Used

🐍 Python
🐼 Pandas
📈 NumPy
📊 Matplotlib
🤖 Scikit-learn

Steps in the Project

  1. Import required libraries
  2. Create dataset with employee information
  3. Filter IT department data
  4. Define features (Experience) and target variable (Salary)
  5. Split data into training and testing sets (80/20)
  6. Train Linear Regression model
  7. Evaluate model performance
  8. Visualize regression line with scatter plot
  9. Predict salary for new experience values
